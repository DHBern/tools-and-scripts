PURPOSE

These Matlab functions (R2021b) allow to read from and write to Transkribus via the REST API, as well as modify the files, e.g., using regular expressions.


CONTENTS

transkribus_io - Showcases the interfacing process.
transkribus_col74823_doc514186_page1.xml - Sample Transkribus file.
bar_sar - Search and replace terms in downloaded Transkribus XML files.
bar_upload - Upload files to Transkribus.
bar_check - Search for files with specific text patterns.
